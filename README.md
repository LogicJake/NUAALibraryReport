# NUAALibraryReport

### 数据来源
之前的图书馆官方推出的南航图书馆个人报告有漏洞，通过修改链接中的学号可以查看其他人的报告，爬虫遍历。
http://librep.nuaa.edu.cn/u5610965.viewer.maka.im/viewer/5610965/T_C7A3A6HT.html?xuehao=学号
